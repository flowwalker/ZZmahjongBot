## 数据管线

- **预处理**：`preprocess_mp.py` — 10 核多进程，288× 物理增强落盘
- **训练加载**：`dataset.py` — `np.concatenate` 一次性加载所有 npz 到连续内存
- **增强**：`augment.py` — 完整的 288-fold 群论变换 (Z₂×S₃×S₃×Z₄ = 2×6×6×4)
- **int8 编码**：WALL×21 / TILE_FROM×3 / HAND_SIZE×14 → int8 → np.savez_compressed
